Impacto da Inteligência Artificial no Mercado de Trabalho (2024–2030)

Objetivo:
Este projeto tem como objetivo utilizar o dataset “AI Impact on Job Market (2024–2030)”, disponível no Kaggle, para desenvolver um modelo preditivo capaz de classificar o impacto da Inteligência Artificial (IA) nas profissões.
O modelo é construído utilizando uma Rede Neural Artificial do tipo Multilayer Perceptron (MLP), implementada com a biblioteca Scikit-learn, buscando prever se uma profissão tende a crescer ou diminuir com o avanço da IA.

Autor:
Claudio A. C. Junior

Contexto:
A Inteligência Artificial tem transformado significativamente a forma como as pessoas trabalham e como as empresas operam. Tarefas que antes eram realizadas manualmente estão sendo automatizadas, impactando diretamente diversas profissões.
Este projeto analisa esse impacto no período de 2024 a 2030, utilizando técnicas de Machine Learning para compreender e prever tendências no mercado de trabalho. Além do desenvolvimento do modelo preditivo, o projeto contribui para uma melhor compreensão de como as novas tecnologias afetam diferentes áreas profissionais.

Etapas do Projeto:

Introdução

Importação e leitura dos dados

Análise exploratória dos dados (EDA)

Pré-processamento dos dados

Modelagem com Rede Neural MLP

Avaliação do modelo

Conclusões e recomendações

Tecnologias Utilizadas

Python

Pandas

NumPy

Scikit-learn

Matplotlib / Seaborn

Resultados Esperados:

Identificar padrões no impacto da IA sobre diferentes profissões

Classificar profissões com tendência de crescimento ou declínio

Apoiar análises e decisões relacionadas ao futuro do mercado de trabalho

Observações
